# Compute Shader

GPU를 사용하여 대량의 일반 데이터 연산을 초고속으로 처리하고 

**GPGPU**기술을 개발자가 쓸 수 있게 **쉐이더** 형태로 제공하는 **프로그램**

참고용(GPGPU의 설명을 보고 오는 것이 좋음)

[CUDA](https://app.notion.com/p/CUDA-389a94a4495380c39a7fe3dd7936fbe7?pvs=21)

## shader와 Compute Shader의 차이

- shader
    - 정해진 그래픽스 파이프 라인 안에서만 작동
    - CPU가 넘겨준 데이터를 화면에 출력하는 단방향 흐름을 가짐
- Compute Shader
    - 그래픽스 파이프라인과 별개로 완전히 독립적임
    - 렌더링을 하지 않아도 실행할 수 있음
    - GPU의 수청 개 코어를 이용해서 대량의 데이터를 계산한 뒤
        
        결과를 CPU로 반환하거나 다른 셰이더로 넘겨줄 수 있음
        

## 스레드 그룹(Thread group)

스레드 : 가장 작은 계산 단위

스레드 그룹 : 스레드를 모아 놓은 상자

## 게임 개발에서 활용도

1. 대규모의 파티클 시스템
2. 대규모 인공지능 및 군중 시뮬레이션
3. 레이트레이싱 및 화면 후 처리
