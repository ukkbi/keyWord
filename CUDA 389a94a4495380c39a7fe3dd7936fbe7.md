# CUDA

## (compute Unified Device Architecture)

그래픽 연산 뿐 아니라 일반적인 프로그래밍 연산도 같이 할 수 있게 해주는 기술

### 등장배경

과거의 GPU는 게임 명령어 외엔 인식하지 못했지만 GPU 내부의 수천개의 코어를 

일반 연산에 활용하는 GPGPU(General-Purpose computing on GPUs)라는 개념을 만들었다  

초기엔 일반 계산 코드를 억지로 3D 그래픽 명령어(셰이더 언어)로 바꾸어 입력해야 하는 문제가 있었다

이를 해결하기 위해 만든 것이 엔비디아에서 출시한 CUDA다 

셰이더 언어가 아닌 C, C++, python 등의 프로그래밍 언어로 직접 제어할 수 있게 만들어줬다

### 작동방식

1. CPU가 병렬처리 구간을 만나면 CUDA를 통해 GPU로 넘겨준다
2. GPU 내부의 CUDA 코어가 데이터를 수천 개로 쪼개어 계산
3. GPU의 계산이 끝난다면 CUDA가 결과 값을 다시 CPU와 메인메모리로 반환함

## CUDA 와 CUDA코어의 차이점

### CUDA

GPU를 계산기처럼 쓸 수 있도록 제어하는 기술과 소프트웨어 환경 전체를 뜻함

### CUDA 코어

엔비디아 GPU에 장착되어 있는 물이적인 연산 장치를 뜻함
