# CPU(CentralProcessingUnit)

## 중앙처리장치

컴퓨터가 수행하는 명령을 해석하고 계산하고 제어하는 역할을 한다

1. 제어
2. 연산
3. 저장

### 1. 제어

컴퓨터의 모든 부품에 명령을 내리는 역할

각 부품은 스스로 작동할 수 없기 때문에 명령을 내리는 CPU의 존재가 중요함

#### 과정

1. 메모리에서 명령어 코드를 가져와 의미를 분석
2. 분석한 결과를 바탕으로 역할을 수행해야 하는 부품에 전기적 신호를 전달
3. 프로그램이 순서대로 실행되도록 어떤 명령어를 가져와야 하는지 경로 지정

### 2. 연산

수학적 계산 및 논리 연산을 수행하는 데이터 처리 과정

핵심 부품으론 ALU(산술노리연산장치), 레지스터, 내부 버스로 구성되어 있음

1. ALU
    
    계산을 처리하는 계산기의 역할을 수행
    
2. 레지스터
    
    CPU 내부의 메모리 게산의 데이터의 일시적 보관 및 결과 기록
    
3. 3. 내부버스
    
    CPU 내부의 데이터 통로
    

#### 과정

1. 메모리의 데이터를 레지스터에 저장
2. 레지스터에 저장된 데이터를 ALU에 입력
3. 입력된 값을 ALU에서 계산
4. 계산 결과를 다시 레지스터의 저장

### 3. 저장

계산 중인 데이터의 보관 및 계산된 결과의 전달

핵심 부품으론 레지스터, 캐시 메모리가 있다

1. 레지스터
    
    위와 동일
    
2. 캐시메모리
    
    CPU와 RAM 사이의 속도 차이를 메꾸기 위한 버퍼 공간 다음에 쓸 것 같은 데이터를 RAM에서 
    
    미리 가져와 저장해 놓는 곳
    

#### 과정

1. 실행할 명령어와 계산에 쓰일 데이터를 ALU에 전달 대기
2. 계산 중간의 결과 값을 일시적으로 보관
3. 계산 결과를 저장 후 활용

# GPU(GraphicsProcessingUnit)

## 그래픽 처리 장치

CPU랑은 달리 명령을 내리기 보단 엄청난 양의 사칙연산을 병렬 연산 후 화면에 반영하는  장치

#### 역할

1. 캐릭터의 움직임, 물리 효과, 이펙트, 빛 반사 등을 실시간으로 계산 후 화면에 반영
2. 병렬처리에 특화되어 있기에 AI의 딥러닝 연산에 활용된다
3. 방대한 데이터의 시뮬레이션 연산

#### 부품

코어

CUDA 코어 : 기초적인 사칙연산을 담당하는 코어

텐서 코어 : *행렬 연산만을 초고속으로 처리하는 코어

*행렬 연산 :  대량의 숫자 데이터를 표 형태로 묶어 한번에 계산하는 방식

RT 코어 : 빛이 반사되는 경로를 실시간으로 계산하는 빛 추적 전용 코어

VRAM : 방대한 데이터양을 대비한 GPU 전용 메모리, 데이터의 임시 보관 역할

기가 스레드 엔진 : 코어들에 일을 공평하고 효율적으로 분배해주는 역할

ROP(RasterOperationsPipeline)

계산된 데이터를 2D 픽셀 데이터로 변환(레스터화)해서 안티앨리어싱(이미지를 전체적으로 다듬는 과정) 처리하고 최종 색상을 결정해서 화면에 반영하는 역할
